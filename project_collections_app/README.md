# Flutter Accordion Collections

This project displays a list of product collections using an
accordion-style UI built with Flutter.

## Features
- Scrollable list of collection cards
- Horizontal preview images
- +N overlay when more than 4 images are available
- Expand / collapse accordion behavior
- Smooth animations
- Clean and reusable widget structure

## Tech Stack
- Flutter
- Dart

## Project Structure
lib/
- models/collection.dart
- screens/collections_screen.dart
- widgets/collection_card.dart

## How to Run
flutter pub get  
flutter run
